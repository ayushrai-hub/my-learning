## Career Progression Framework

### L1: Junior Engineer (0-2 years)
**Characteristics**: Fundamentals, guided work, single component ownership
- Execute tasks with supervision
- Basic understanding of tools and processes
- Solid grasp of core concepts and syntax
- Ability to debug simple issues
- Contribute to documentation and testing
- Time to L2: 18-24 months

### L2: Mid-Level Engineer (2-4 years)
**Characteristics**: Feature ownership, system understanding, mentoring juniors
- Own end-to-end feature development
- Understand system architecture and trade-offs
- Debug complex issues independently
- Mentor newer team members
- Contribute to process improvements
- Time to L3: 24-36 months

### L3: Senior Engineer (4-7 years)
**Characteristics**: System design, tech leadership, cross-team impact
- Design scalable systems and architectures
- Lead technical initiatives and decisions
- Optimize performance and reliability
- Drive engineering excellence initiatives
- Influence broader technical strategy
- Time to L4: 30-42 months

### L4: Staff Engineer (7-10 years)
**Characteristics**: Architecture, org-wide influence, strategic technical decisions
- Architect complex, multi-service systems
- Influence organizational technical direction
- Solve ambiguous, high-impact problems
- Drive architectural modernization efforts
- Mentor senior engineers and technical leads
- Time to L5: 36+ months

### L5: Principal/Distinguished Engineer (10+ years)
**Characteristics**: Company-wide technical vision, industry impact
- Shape company technical strategy and vision
- Drive industry-wide innovation and standards
- Lead large-scale technical transformations
- Build high-performing engineering organizations
- Influence technology industry directions

## A. Computer Science Fundamentals

### Why This Matters
Computer Science fundamentals form the theoretical foundation for all engineering work, enabling you to make informed decisions about algorithms, data structures, and system design regardless of your specific domain.

### Learning Progression

#### üéì Foundation (L1 - Junior)
**Goal:** Build mathematical and logical reasoning skills for problem-solving
**Core Concepts:**
- Variables, loops, conditionals (prerequisite for all programming)
- Basic data types and memory management (understanding stack vs heap)
- Functions and parameter passing (value vs reference semantics)
- Basic debugging techniques (print statements, breakpoints)

**Essential Tools:**
- Python/Java/C++ - Basic syntax and IDE setup
- Git/GitHub - Version control basics

**Practical Projects:**
- Build a simple calculator with basic arithmetic operations
- Implement a text-based Tic-Tac-Toe game
- Create a program that processes and analyzes a CSV file

#### üî® Intermediate (L2 - Mid-Level)
**Goal:** Master core data structures and algorithms with implementation experience
**Core Concepts:**
- Arrays, linked lists, stacks, queues (builds on: basic data types)
- Binary trees, heaps, hash tables (data structure trade-offs)
- Sorting algorithms: quicksort, mergesort, heapsort (time/space complexity analysis)
- Graph traversal algorithms: DFS, BFS (problem-solving patterns)

**Essential Tools:**
- LeetCode/HackerRank - Algorithm practice platforms
- Python data science stack (NumPy, Pandas) - Array manipulation

**Practical Projects:**
- Implement a complete binary search tree with insert/delete operations
- Build a web crawler that respects robots.txt and rate limiting
- Create an in-memory database with basic CRUD operations

#### üöÄ Advanced (L3 - Senior)
**Goal:** Optimize algorithms and understand system-level implications
**Core Concepts:**
- Dynamic programming patterns (memoization, tabulation)
- Advanced graph algorithms (Dijkstra, topological sort, MST)
- Bit manipulation techniques (efficient space usage)
- Algorithm complexity analysis in distributed systems

**Production Skills:**
- Time-space trade-offs in real systems
- Cache-friendly data structures
- Memory-efficient algorithms for large datasets

**Practical Experience:**
- Optimize a distributed system for geographic data processing
- Implement a real-time analytics system handling millions of events
- Build a scalable recommendation engine using collaborative filtering

#### ‚≠ê Expert (L4-L5 - Staff+)
**Goal:** Innovate algorithms for novel problems and lead architectural decisions
**Deep Expertise:**
- Advanced optimization techniques (constant factor improvements)
- Probabilistic data structures (Bloom filters, HyperLogLog)
- Algorithm design for specialized hardware (GPU, TPU)
- Security considerations in algorithm implementation

**Strategic Skills:**
- Choosing algorithms for business-critical systems
- Balancing algorithmic complexity with maintainability
- Contributing to open-source algorithm libraries
- Teaching advanced algorithms to senior engineers
