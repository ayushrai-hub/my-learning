- **GitHub Student Developer Pack**
  - *Includes:* Private repos, GitHub Pro, $200 DigitalOcean credit, AWS Educate, JetBrains IDEs
  - *Requirements:* Valid student email or documentation

  - *Products:* IntelliJ IDEA, PyCharm, WebStorm, GoLand, etc.
