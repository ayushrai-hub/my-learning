# Human & Team Skills

- **Analytical & Systems Thinking**

  - Problem decomposition: breaking down complex problems, root cause analysis
  - Systems perspective: understanding interconnections, feedback loops, emergent behavior
  - Critical thinking: evidence evaluation, assumption questioning, logical reasoning
  - Pattern recognition: identifying recurring themes, anti-patterns, solution patterns
  - Mental models: frameworks for understanding, decision-making heuristics, biases
  - Complexity management: dealing with uncertainty, ambiguity, trade-offs
  - Strategic thinking: long-term planning, vision alignment, priority setting

- **Technical Communication & Writing**

  - Technical writing: clarity, conciseness, audience awareness, structure, documentation
  - RFC culture: request for comments, design documents, decision records, collaboration
  - ADR documentation: architectural decision records, context, consequences, alternatives
  - Presentation skills: technical presentations, storytelling, visual aids, engagement
  - Code documentation: inline comments, API documentation, examples, maintenance
  - Status communication: progress updates, blockers, risk communication, transparency
  - Cross-functional communication: business stakeholders, non-technical audiences

- **Mentoring & Knowledge Transfer**

  - Mentoring skills: active listening, guidance, feedback, goal setting, support
  - Knowledge sharing: documentation, presentations, workshops, pair programming
  - Teaching techniques: explaining complex concepts, adapting to learning styles
  - Feedback delivery: constructive feedback, specific examples, growth-oriented
  - Code reviews: educational reviews, knowledge transfer, best practice sharing
  - Onboarding: new team member integration, knowledge transfer, culture transmission
  - Succession planning: knowledge documentation, skill development, transition planning

- **Collaboration & Leadership**

  - Team dynamics: psychological safety, trust building, conflict resolution, inclusion
  - Negotiation skills: win-win outcomes, stakeholder management, compromise, influence
  - Product mindset: user focus, business value, prioritization, outcome orientation
  - Project management: planning, execution, risk management, stakeholder communication
  - Cross-team collaboration: alignment, dependency management, communication protocols
  - Technical leadership: vision setting, decision making, team guidance, architecture
  - Change management: adoption strategies, resistance handling, communication, training

- **Personal Effectiveness & Growth**
  - Time management: prioritization, focus techniques, productivity systems, boundaries
  - Learning strategies: continuous learning, skill development, knowledge acquisition
  - Growth mindset: embracing challenges, learning from failure, resilience, adaptation
  - Self-awareness: strengths, weaknesses, biases, emotional intelligence, reflection
  - Career development: goal setting, skill building, networking, opportunity creation
  - Work-life balance: stress management, burnout prevention, sustainable practices
  - Innovation mindset: creativity, experimentation, risk-taking, continuous improvement
