# Front-End Engineering (TypeScript, React, WebGL)

- **Modern JavaScript & TypeScript**

  - ES2023+ features: top-level await, private fields, optional chaining, nullish coalescing
  - TypeScript generics: constraints, conditional types, mapped types, template literals
  - Utility types: Partial, Required, Pick, Omit, Record, Exclude, Extract
  - Advanced types: union, intersection, discriminated unions, type guards
  - Module systems: ESM, CommonJS, dynamic imports, tree shaking
  - Build tools: TypeScript compiler, tsconfig.json, project references
  - Type declaration files: .d.ts, DefinitelyTyped, module augmentation

- **React Architecture & Patterns**

  - Component lifecycle: mounting, updating, unmounting, error boundaries
  - Hooks ecosystem: useState, useEffect, useContext, useReducer, custom hooks
  - Context API: provider pattern, context composition, performance considerations
  - Suspense: lazy loading, error boundaries, concurrent features
  - Concurrent rendering: time slicing, priority scheduling, interruption
  - React 18 features: automatic batching, startTransition, useDeferredValue
  - Performance optimization: React.memo, useMemo, useCallback, profiling

- **State Management & Architecture**

  - State machines: XState, finite state machines, statecharts, guards/actions
  - Global state: Zustand, Redux Toolkit, Jotai, atomic state management
  - Local state patterns: compound components, render props, custom hooks
  - Data fetching: React Query, SWR, Apollo Client, cache management
  - Form management: React Hook Form, Formik, validation strategies
  - State normalization: entity relationships, selectors, derived state
  - Immutable updates: Immer, immutability helpers, performance implications

- **Next.js & Full-Stack React**

  - App Router: file-based routing, layouts, loading states, error handling
  - Server components: RSC, streaming, selective hydration, performance benefits
  - API routes: serverless functions, middleware, authentication, database integration
  - Static generation: SSG, ISR, dynamic routes, build-time optimization
  - Server-side rendering: SSR, hydration, performance considerations
  - Edge runtime: Vercel Edge, Cloudflare Workers, geographic distribution
  - Deployment: Vercel, Netlify, self-hosted, CI/CD integration

- **Performance & Core Web Vitals**

  - Largest Contentful Paint (LCP): image optimization, critical rendering path
  - Cumulative Layout Shift (CLS): layout stability, font loading, image dimensions
  - Interaction to Next Paint (INP): event handling, main thread blocking, optimization
  - Performance monitoring: Web Vitals API, Real User Monitoring, synthetic testing
  - Bundle optimization: code splitting, lazy loading, tree shaking, compression
  - Image optimization: WebP, AVIF, responsive images, lazy loading
  - Font optimization: font-display, preloading, variable fonts, FOUT/FOIT

- **Accessibility & Inclusive Design**

  - WCAG 2.2 guidelines: perceivable, operable, understandable, robust
  - ARIA roles: landmarks, widgets, live regions, properties and states
  - Semantic HTML: proper element usage, heading hierarchy, form labels
  - Keyboard navigation: focus management, tab order, skip links, shortcuts
  - Screen reader testing: NVDA, JAWS, VoiceOver, announcement patterns
  - Color accessibility: contrast ratios, color blindness, high contrast mode
  - Motion accessibility: prefers-reduced-motion, vestibular disorders

- **Design Systems & Component Libraries**

  - Storybook: component documentation, visual testing, addon ecosystem
  - Design tokens: colors, typography, spacing, semantic naming, tooling
  - Component API design: props, composition, polymorphism, extensibility
  - Theme systems: CSS custom properties, context providers, dark mode
  - Icon systems: SVG optimization, icon fonts, sprite sheets, accessibility
  - Layout systems: CSS Grid, Flexbox, container queries, responsive design
  - Testing: visual regression, accessibility testing, interaction testing

- **WebGL & 3D Graphics**
  - Three.js fundamentals: scene, camera, renderer, geometry, materials
  - 3D mathematics: vectors, matrices, quaternions, transformations
  - GLSL shaders: vertex shaders, fragment shaders, uniforms, attributes
  - Lighting models: ambient, directional, point lights, shadows, PBR
  - Texture mapping: UV coordinates, mipmaps, filtering, compression
  - Performance optimization: LOD, frustum culling, batching, instancing
  - WebXR integration: VR/AR experiences, hand tracking, spatial computing
- ES 2023+, strict TS generics & utility types
- React hooks, context, suspense, concurrent rendering
- State-machines (XState, Zustand); Next.js App Router
- Core Web Vitals (LCP, CLS, INP)
- Accessibility (WCAG 2.2, ARIA roles)
- Design systems (Storybook, design tokens)
- Three.js basics, GLSL shaders
