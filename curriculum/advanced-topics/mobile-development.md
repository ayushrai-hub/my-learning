# Mobile & Cross-Platform Development

- **Flutter Development**

  - Widget architecture: stateless, stateful widgets, widget tree, composition patterns
  - State management: Provider, Riverpod, BLoC, Redux, state architecture patterns
  - Navigation: Navigator 2.0, route management, deep linking, navigation patterns
  - Platform integration: platform channels, native code integration, plugin development
  - Performance optimization: widget rebuilds, rendering pipeline, memory management
  - Testing: unit tests, widget tests, integration tests, golden tests, test automation
  - Deployment: app store deployment, code signing, release management, CI/CD

- **React Native Development**

  - Component architecture: functional components, hooks, lifecycle, performance
  - Navigation: React Navigation, stack navigation, tab navigation, drawer navigation
  - State management: Redux, Context API, Zustand, state patterns, data flow
  - Native modules: bridging, native code integration, platform-specific functionality
  - Performance: JavaScript bridge, native performance, optimization techniques
  - Debugging: Flipper, React Native Debugger, performance profiling, crash reporting
  - Code push: over-the-air updates, deployment strategies, rollback procedures

- **Cross-Platform Architecture**
  - Code sharing: shared business logic, platform-specific UI, architecture patterns
  - Platform abstraction: abstraction layers, dependency injection, platform services
  - Asset management: platform-specific assets, resolution handling, optimization
  - Configuration management: environment variables, feature flags, platform configs
  - Build systems: platform-specific builds, shared tooling, automation, CI/CD
  - Testing strategies: shared test suites, platform-specific tests, device testing
  - Maintenance: code synchronization, platform updates, dependency management
