# Accessibility & Inclusive Design

- **WCAG Compliance & Standards**

  - WCAG 2.2 guidelines: perceivable, operable, understandable, robust principles
  - Conformance levels: A, AA, AAA requirements, legal compliance, business benefits
  - Success criteria: specific requirements, testing methods, implementation guidance
  - Accessibility audits: automated testing, manual testing, user testing, expert review
  - Legal requirements: ADA compliance, Section 508, international standards, litigation risks
  - Documentation: accessibility statements, compliance reports, remediation plans
  - Continuous monitoring: ongoing testing, regression prevention, improvement tracking

- **Semantic HTML & ARIA**

  - Semantic elements: proper element usage, document structure, meaning conveyance
  - ARIA roles: landmark roles, widget roles, document structure roles, live regions
  - ARIA properties: labels, descriptions, states, relationships, dynamic content
  - Focus management: tab order, focus indicators, skip links, keyboard navigation
  - Form accessibility: labels, error messages, instructions, validation, fieldsets
  - Table accessibility: headers, captions, summaries, complex table navigation
  - Media accessibility: alt text, captions, transcripts, audio descriptions

- **Assistive Technology Support**

  - Screen readers: NVDA, JAWS, VoiceOver, testing procedures, user experience
  - Voice control: Dragon NaturallySpeaking, voice navigation, command recognition
  - Switch devices: switch navigation, scanning patterns, timing considerations
  - Magnification software: screen magnifiers, zoom functionality, layout considerations
  - Cognitive assistance: reading assistance, memory aids, attention support
  - Motor impairments: alternative input methods, gesture alternatives, timing adjustments
  - Testing procedures: assistive technology testing, user feedback, iterative improvement

- **Visual Design & Accessibility**

  - Color contrast: WCAG contrast ratios, color blindness considerations, testing tools
  - Typography: readable fonts, font sizes, line spacing, text scaling, readability
  - Layout design: consistent navigation, predictable layouts, white space, visual hierarchy
  - Motion sensitivity: prefers-reduced-motion, animation controls, vestibular disorders
  - Visual indicators: focus indicators, error states, status communication, iconography
  - Responsive design: mobile accessibility, touch targets, orientation support
  - Dark mode: contrast considerations, user preferences, system integration

- **Internationalization & Localization**
  - Language support: multilingual content, language switching, translation quality
  - Text direction: RTL support, bidirectional text, layout adjustments, text alignment
  - Cultural considerations: color meanings, imagery, symbols, cultural sensitivity
  - Date/time formats: locale-specific formats, timezone handling, calendar systems
  - Number formats: currency, decimals, thousands separators, locale conventions
  - Input methods: international keyboards, IME support, character encoding
  - Content adaptation: cultural adaptation, local regulations, market requirements
